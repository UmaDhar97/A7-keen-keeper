# Keen Keeper

A modern React-based sports/team management web application where users can explore player information, manage team selections, and interact with dynamic UI components.

## Live Link
https://orange-bar-afa4.umadhar97.workers.dev/

## Technologies Used

- React.js
- Vite
- Tailwind CSS
- DaisyUI
- React Router DOM

## Key Features

- Dynamic player/team card rendering
- Responsive modern UI design
- Interactive routing and navigation

## Additional Features

- Reusable component architecture
- Optimized asset management
- Clean folder structure for scalability
